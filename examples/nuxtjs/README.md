# Vitest / Nuxt 3 / Vue 3

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn

# npm
npm install
```

And then to launch Nuxt at least on time to generate .nuxt directory:

```bash
# yarn
yarn dev

# npm
npm run dev
```

## Launch the tests

```bash
yarn test
```

```bash
npm run test
```

Other scripts are available: test:verbose and test:watch.
